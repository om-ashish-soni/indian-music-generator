# Raga Reference

A curated map of Hindustani classical ragas to mood, time-of-day, and recommended
masters. This is the listening side of the skill — for actual spiritual sessions,
play these via [`play-raga.sh`](./play-raga.sh).

## Time-of-day cycle

Hindustani classical assigns each raga a `samay` (time). Listening at the right
hour is part of the tradition.

| Time            | Mood                  | Ragas                                                                |
| --------------- | --------------------- | -------------------------------------------------------------------- |
| Dawn (4-7 AM)   | Devotional, awakening | **Bhairav**, Ahir Bhairav, Lalit, Ramkali                            |
| Morning (7-10)  | Energetic, bright     | **Todi**, Bilaskhani Todi, Gujari Todi, Asavari                      |
| Late morning    | Pastoral              | **Desh**, Khamaj, Tilang                                             |
| Afternoon (12-4)| Contemplative         | **Bhimpalasi**, Multani, Madhuvanti                                  |
| Evening (6-9)   | Romantic, peaceful    | **Yaman**, Yaman Kalyan, Puriya Dhanashree, Marwa, Hamir             |
| Late evening    | Meditative            | **Bageshree**, Hansadhwani                                           |
| Night (9-12)    | Deep, introspective   | **Malkauns**, Darbari Kanada, Chandrakauns, Kaunsi Kanada            |
| Late night      | Resolution            | **Bhairavi** (also used to close any concert regardless of time)     |

## Mood index

- **Spiritual / devotional**: Bhairav, Bhairavi, Malkauns, Ahir Bhairav
- **Calm / meditative**: Yaman, Bhimpalasi, Bageshree, Malkauns
- **Romantic**: Yaman, Khamaj, Tilak Kamod
- **Melancholic / longing**: Marwa, Darbari, Todi, Puriya Dhanashree
- **Joyful**: Khamaj, Desh, Pahadi, Kafi

## Masters by instrument

### Sitar
- **Pandit Ravi Shankar** — the global ambassador; vast catalogue across ragas
- **Ustad Vilayat Khan** — vocal-influenced (gayaki ang), incredibly expressive
- **Pandit Nikhil Banerjee** — pure, emotional, Maihar gharana
- **Anoushka Shankar** — modern voice, fusion-friendly while traditional

### Bansuri (bamboo flute)
- **Pandit Hariprasad Chaurasia** — defining voice of the modern bansuri
- **Pandit Pannalal Ghosh** — pioneer who established bansuri as classical solo
- **Rakesh Chaurasia** — Hariprasad's nephew, contemporary master

### Sarod
- **Ustad Ali Akbar Khan** — towering figure, deep emotional range
- **Ustad Amjad Ali Khan** — graceful, melodic
- **Pandit Buddhadev Dasgupta** — Shahjahanpur gharana, scholarly

### Santoor
- **Pandit Shivkumar Sharma** — single-handedly elevated santoor to classical solo

### Sarangi
- **Ustad Sultan Khan** — vocal-quality bowing
- **Pandit Ram Narayan** — established sarangi as solo instrument

### Vocal (for raga study)
- **Pandit Bhimsen Joshi** — Kirana gharana, authoritative
- **Pandit Jasraj** — Mewati gharana, devotional intensity
- **Ustad Amir Khan** — slow alap mastery, deeply meditative

## Essential albums for spiritual listening

- **"Call of the Valley"** (1967) — Shivkumar Sharma, Hariprasad Chaurasia, Brijbhushan Kabra. The defining calm-classical album.
- **"Three Ragas"** (1956) — Ravi Shankar's first international release.
- **"Aman Aman: Live at Carnegie Hall"** — Hariprasad Chaurasia.
- **"Inside the Kremlin"** — Ravi Shankar's meditative Moscow concert.
- **"The Master Musicians of India"** — Ali Akbar Khan compilations.

## Vocabulary cheat-sheet

| Term      | Meaning                                                                |
| --------- | ---------------------------------------------------------------------- |
| Sa Re Ga Ma Pa Dha Ni | The seven swaras (notes) — `Sa` = tonic, like Western Do  |
| Vadi      | Most prominent note in the raga                                        |
| Samvadi   | Second-most prominent (usually a 4th or 5th from vadi)                 |
| Aroha     | Ascending note pattern                                                 |
| Avaroha   | Descending note pattern                                                |
| Pakad     | Signature short phrase that identifies the raga                        |
| Alap      | Slow, free-rhythm exploration of the raga (no tabla)                   |
| Jor       | Medium-tempo section after alap, with pulse but no tabla cycle yet     |
| Jhala     | Fast climactic section, rapid plucking                                 |
| Gat       | Composed melodic theme played in a tala (rhythmic cycle)               |
| Taala     | Rhythm cycle — Teentaal (16), Jhaptaal (10), Rupak (7), Ektaal (12)    |
| Meend     | Pitch glide between notes (key sitar/sarangi expression)               |
| Gamak     | Heavy oscillation around a note                                        |
| Tanpura   | Drone instrument played throughout — anchors Sa and Pa                 |
| Vilambit  | Slow tempo                                                             |
| Madhya    | Medium tempo                                                           |
| Drut      | Fast tempo                                                             |
